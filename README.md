# Spinning Torus

Saw this on yt originally, decided to implement.

Video: https://www.youtube.com/watch?v=DEqXNfs_HhY&t=6s

## Summary:
- Generate Torus (ring of circles)
  - Iterate over the big circle
  - Generate points of circle around the current point (I stored normal as well)
- For each frame, iterate over all points on the torus, and rotate each using a rotational matrix
- Do some math and figure out position on the 'screen' in front of camera (everything's linear)
- If the point is on screen, figure out how much light you want.
  - Use z-buffering to check whether something is alr in front
  - Calculate dot product of normal with whatever side you want the light to be on (e.g. for above and towards camera, use (0, 1, -1))
